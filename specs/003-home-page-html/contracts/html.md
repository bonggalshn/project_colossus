# HTML Page Contract

## Public Interface

### File Structure

| File | Required | Description |
|------|----------|------------|
| index.html | Yes | Main HTML document |
| assets/css/style.css | Yes | Stylesheet |

### HTML Requirements

- Valid HTML5 document structure
- Semantic HTML elements (header, nav, main, footer)
- meta viewport for responsive design
- lang attribute on html element

### Accessibility

- alt text on all images
- ARIA labels where needed
- Keyboard navigable
- Focusable elements
- WCAG 2.1 AA contrast ratio

### Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
- Opera (latest)
